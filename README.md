QuickBite

QuickBite is a responsive food delivery web application built using HTML, CSS, and JavaScript.
It dynamically loads product data from a JSON file and provides interactive cart functionality.
____________________________________________________________________________________________________________________________________________________________________
🚀 Features

Responsive UI

Dynamic product rendering (JSON-based)

Add to Cart

Remove from Cart

Quantity management

Real-time total price calculation

🛠 Tech Stack

HTML5

CSS3

JavaScript (ES6)

JSON
